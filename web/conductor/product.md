# Initial Concept

A financial planning and simulation tool for developers, similar to ProjectionLab but built with a developer-first approach. It integrates with AI via skills, MCP, and a CLI, allowing for code-first configuration and git-like branching for scenario testing.

# Product Guide: Prosper

## 1. Overview
Prosper is a web-based financial planning and simulation tool that lets users plan their financial life through a dynamic, git-inspired branching tree. It's designed for developers and tech-savvy individuals who want more flexibility and control over their financial models than traditional tools offer.

## 2. Target Audience
- **Developers:** Users who value structured planning, code-inspired workflows, and powerful visualization.
- **Finance Tech Enthusiasts:** Individuals seeking a modern, flexible alternative to monolithic financial tools.

## 3. Core Features
- **Git-Inspired Scenario Branching:** Visualize your financial future as a node-based tree. Create new branches for different life choices (e.g., career changes, major purchases) and compare outcomes across multiple parallel scenarios.
- **Dynamic Node-Based UX:**
    - Root Node: Representing the present financial state.
    - Action Nodes: Future events (e.g., "Change Career") that affect downstream outcomes.
    - Branching: Easily fork the main branch to test new scenarios.
- **Integrated Goal Tracking:**
    1. **Fixed Date Goals:** (e.g., Retirement) Visualized as steady progress markers across all branches.
    2. **Action-Based Goals:** Event-specific nodes that can be added or removed to test their impact.
    3. **Measured Goals:** Triggered automatically when specific financial conditions (e.g., reaching a certain net worth) are met.

## 4. User Experience
- **Graphical Branching UI:** A modern tree interface mimicking a git graph, with nodes sorted by date and branches clearly defined.
- **Interactive Scenarios:** Add, remove, or modify nodes in any branch to see real-time updates to your financial projections.
- **Outcome Summaries:** A detailed table below the tree summarizing the financial results of the selected branch.

## 5. Vision
Prosper aims to become a full-scale platform, evolving from its initial web focus to eventually include deep AI integration (MCP, skills) and a command-line interface for the ultimate developer-first financial experience.
