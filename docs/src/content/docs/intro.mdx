# Introduction

Welcome to the documentation for **Prosper**.

Prosper is fundamentally a ProjectionLab for developers. We believe financial planning should not be a static, once-a-year exercise with complex spreadsheets. It should be dynamic, testable, and version-controlled.

## The Problem

Traditional financial planning tools either abstract away too much detail (making them useless for complex scenarios) or require you to learn proprietary formulas and cumbersome UI.

## The Solution

Prosper brings developer workflows to personal finance:

-   **Node-Based System:** Every financial event is a node in a tree. Add an income node, connect it to an expense node, and branch it off into investment nodes.
-   **Git-like Branching:** What happens if you get a 10% raise? Create a branch, change the value, and compare the two timelines.
-   **AI Integration:** Future versions of Prosper will connect directly to AI agents to help you optimize the nodes based on your financial goals.

Start exploring your financial future with the precision of a software engineer.
