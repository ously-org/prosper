# Welcome to Prosper

Prosper is a financial planning and simulation tool designed for developers. Think of it as a "ProjectionLab for developers" using a node-based, git-like branching system for financial scenario testing.

## Why Prosper?

Traditional financial tools are opaque and inflexible. Prosper is built for developers who want to apply the rigors of software engineering—like branching, version control, and node-based logic—to their personal finances.

### Quick Start

1.  **Define a Node:** Start by creating a node that represents a financial goal or an action (like "Buy a House" or "Invest in S&P 500").
2.  **Branch Out:** Create a new branch to simulate different outcomes without affecting your main financial model.
3.  **Track and Adapt:** As your real-world situation changes, update your nodes and watch the tree adapt automatically.

Get started by defining your first financial node today.
